Modified SQL Database - March 2023

As part of a class on Data Structures and Algorithms, I was tasked with constructing a custom SQL database, implemented in C++, to meet a set of performance and functionality specifications.
Important aspects of this project include effecient searching, printing, deleting, and joining of tables. Tables are implemented using the standard library hash tables, which enables sorting and filtering using binary tree indices to optimize search algorithms, making deletion or output of specific rows much faster.
A unique aspect of this custom SQL implementation is the ability to join two tables together. This functionality gives the opportunity to find the intersection of two datasets, and prints the specified rows and values to the terminal. There is a demo of these functionalities below.
