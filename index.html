Modified SQL Database - March 2023

As part of a class on Data Structures and Algorithms, I was tasked with constructing a custom SQL database, implemented in C++, to meet a set of performance and functionality specifications.
Important aspects of this project include effecient searching, printing, deleting, and joining of tables. Tables are implemented using the standard library hash tables, which enables sorting and filtering using binary tree indices to optimize search algorithms, making deletion or output of specific rows much faster.
A unique characteristic of this custom SQL implementation is the ability to join two tables together. This functionality gives the opportunity to find the intersection of two datasets, and prints the specified rows and values to the terminal. There is a demo of these functionalities below.

Digital Signal Processor Project Team - March 2022

As part of an introductory engineering course, I put together a team of 4 students to design and implement a Digital Signal Processor, which accomplished a task of our choice.
We came up with a few ideas, and in order to make sure everyone was happy with our project, I came up with an idea of how to combine all of our ideas into one project.
Our project involved programming a synthesizer, transcriber, and a "teacher" of sorts. The synthesizer allows users to play music directly from the UI, with two octaves of notes, and their choice of note duration. The transcriber takes in any given audio file, and display the pitches on a stem plot.
The teaching software displays the pitches of a song, and the user would play the displayed song. Next, the teaching software would calculate the user's accuracy, and give them a score of how accurately they played the music.
